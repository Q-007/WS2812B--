## 硬件要求

- ESP32开发板
- WS2812B LED灯带（64个灯珠，8x8排列）
- 5V电源（推荐至少2A）

## 接线说明

```
ESP32          WS2812B
GPIO 4   ----> DIN (数据输入)
GND      ----> GND
5V       ----> VCC (外部电源)
```

**注意：** 
- WS2812B需要5V供电，大电流时请使用外部电源
- 数据线默认连接GPIO 4，可在代码中修改
- 建议在数据线上串联一个470Ω电阻保护

## 安装步骤

### 1. 安装MicroPython固件

1. 下载MicroPython固件（ESP32）：https://micropython.org/download/esp32/
2. 使用esptool.py刷入固件：

```bash
# 擦除Flash
esptool.py --chip esp32 --port COM3 erase_flash

# 刷入固件
esptool.py --chip esp32 --port COM3 --baud 460800 write_flash -z 0x1000 esp32-xxxxxx.bin
```

### 2. 上传代码

使用Thonny IDE或ampy工具上传文件到ESP32：

#### 使用Thonny IDE
1. 打开Thonny IDE
2. 选择"工具" -> "选项" -> "解释器"
3. 选择"MicroPython (ESP32)"和正确的COM口
4. 在文件浏览器中右键文件，选择"上传到/"

#### 使用ampy
```bash
# 安装ampy
pip install adafruit-ampy

# 上传文件
ampy --port COM3 put boot.py
ampy --port COM3 put main.py
```

### 3. 重启ESP32

重启后程序自动运行。

## 使用说明

### 1. 连接WiFi热点

- SSID: `ESP32_LED_Matrix`
- 密码: **无需密码**（开放网络）

### 2. 访问控制界面

在浏览器中访问：`http://192.168.13.14`

### 3. 控制LED

- **选择颜色**：使用颜色选择器或预设颜色
- **橡皮擦**：点击第一个预设颜色（🧹）进入橡皮擦模式
- **点击绘制**：单击灯珠设置颜色
- **滑动绘制**：按住鼠标/触摸滑动，像画笔一样绘制
- **设置全部灯珠**：将所有灯珠设置为选定颜色
- **彩虹效果**：显示预设的彩虹渐变效果
- **清除所有**：关闭所有灯珠
- **自动播放**：12秒无操作后自动轮播图案

### 4. 图案管理

- **保存图案**：输入名称后点击"保存当前图案"
- **加载图案**：从列表中选择图案并点击"加载"
- **删除图案**：从列表中选择图案并点击"删除"
- **刷新列表**：点击"刷新图案列表"更新显示

## 配置参数

在 `main.py` 中可以修改以下参数：

```python
WIFI_SSID = "ESP32_LED_Matrix"      # WiFi热点名称
WIFI_PASSWORD = None                # None=开放网络，字符串=加密
CUSTOM_IP = "192.168.13.14"         # 自定义IP地址
LED_PIN = 4                         # WS2812B数据引脚
LED_COUNT = 64                      # LED总数
MATRIX_WIDTH = 8                    # 矩阵宽度
MATRIX_HEIGHT = 8                   # 矩阵高度
PATTERNS_FILE = "patterns.json"     # 图案保存文件
```

**IP地址说明**：
- 默认使用 `192.168.13.14`
- 可以修改为任何合法的私有IP地址（每段0-255）
- 例如：`192.168.1.100`, `192.168.10.1` 等
- ⚠️ **不能使用** `192.168.521.1314`（每段不能超过255）

**密码设置**：
- `WIFI_PASSWORD = None` - 开放网络（无需密码）
- `WIFI_PASSWORD = "12345678"` - 设置密码（至少8位）

## 故障排除

### LED不亮
1. 检查接线是否正确
2. 确认电源供电充足（5V，至少2A）
3. 检查GPIO引脚配置是否正确

### 无法连接WiFi
1. 搜索WiFi热点 `ESP32_LED_Matrix`（无需密码）
2. 确认ESP32已启动（查看串口输出）
3. 检查手机/电脑WiFi功能是否正常
4. 尝试重启ESP32

### 网页无法访问
1. 确保已连接到ESP32的WiFi热点
2. 访问自定义IP：`http://192.168.13.14`
3. 检查防火墙设置
4. 尝试清除浏览器缓存

### 图案无法保存
1. 检查ESP32的Flash空间是否充足
2. 查看串口输出的错误信息
3. 尝试删除一些旧图案
4. 重启ESP32后重试

### 图案加载失败
1. ✅ **已在v3.2.2修复**（数据结构不一致问题）
2. 如果仍然失败，删除 `patterns.json` 后重启
3. 检查图案文件是否损坏
4. 查看串口输出的详细错误信息

### 图案顺序混乱
1. ✅ **已在v3.2.2修复**（统一使用列表结构）
2. 图案现在按保存时间先后排列
3. 更新已有图案会保持原位置
4. 新保存的图案会追加到末尾

## 使用技巧 💡

### 滑动绘制
- 按住鼠标左键或触摸屏幕，滑动绘制
- 像使用画笔一样自由创作
- 支持桌面端和移动端

### 橡皮擦工具
- 点击预设颜色第一个（🧹）进入橡皮擦模式
- 橡皮擦模式下点击或滑动即可熄灭灯珠
- 再选择其他颜色退出橡皮擦模式

### 自动播放
- 12秒无操作后自动开始播放保存的图案
- 每个图案显示5秒，带渐入渐出效果
- 任何操作（点击、滑动、按钮）都会停止自动播放
- 自动播放会循环播放所有图案

### 图案管理
- 给图案起个有意义的名字，方便管理
- 定期备份 `patterns.json` 文件
- 可以在电脑上编辑 `patterns.json` 批量修改

### 自定义IP
- 想要 `192.168.521.1314`？
- 虽然这不是合法的IP地址，但你可以使用：
  - `192.168.13.14` （当前使用）
  - `192.168.52.13`
  - `192.168.5.21`
- 记住：IP每段必须是0-255之间的数字
